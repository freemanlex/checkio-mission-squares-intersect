二维平面上的正方形可以定义为一个图元(x, y, l) ，其中(x, y) 是左下角的坐标，l 是正方形的边长（我们只考虑与轴对齐的正方形）。给定两个正方形作为图元(x1, y1, l1) 和(x2, y2, l2) ，您的函数应确定这两个正方形是否相交，即它们的areas 至少有一个共同点，即使这一个点只是这两个正方形放在小猫角时的共同角点。

我们还为您准备了一根大棒和一根胡萝卜：

  • 在编写这个任务时，尽量不要使用任何循环或列表理解，而只使用整数比较和条件语句来计算结果；

  • 实际上，确定两个正方形不相交，然后否定这个答案要容易得多。如果两个正方形中的一个在水平方向上结束于另一个开始之前，或者在垂直方向上发生同样的情况，那么这两个正方形就不相交。
